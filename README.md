# jon's dotfiles

Config files, managed by the [GNU Stow](https://www.gnu.org/software/stow/) symlink farm manager.

## usage

```bash
git clone https://github.com/jon-codes/.dotfiles.git ~/.dotfiles
```

### symlinking individual configs

```bash
cd ~/.dotfiles
```

```bash
stow --dotfiles zsh
```
